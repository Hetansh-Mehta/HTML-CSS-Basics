Styling the Font

We will use the properties of font and text to style the fonts

  • Fonts

    When using fonts on the web, we must first consider what fonts are available to our users. Every operating sustem comes with a set of pre-installed fonts that we can use to customize the website. For a complete list of "web-safe" fonts, follow this link

    The way it works is simple. When using the font-family property, we should specify the font(s) we want to use in the HTML.

    Syntax: font-family: font1, font2, font3, ...;

    Then the browser, starting from left to right, looks at the font(s) we've specified and checks to see if it can render the text using the fonts. If it can't use the first font, then the browser moves to the next font, and so on.

    The purpose of specifying multiple fonts is because not all fonts are available on veery operating system. So, specifying multiply, similar fonts ensures users have a consistent experience regardless of the operating system they are using.

  • Custom Fonts

    It is also possible to use custom fonts on the web. Check out Google Fonts to see some open-source web fonts.
