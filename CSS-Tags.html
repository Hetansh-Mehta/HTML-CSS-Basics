CSS Basics

  • CSS: Cascading Syle Sheets is the markup language that we use to describe how elements on website look. It gives us the ability to change the way elements look.

  • Things that can be controlled:

    1. Colour
    2. Font
    3. Spacing
    4. Layout

  • It lets us separate the description of the content we want on the page (HTML) from the description of how we want the content to look on the page (CSS). Literally all the code that makes the page loog the way it does, is separate from the text, images, and content that is on the page.

CSS RULESETS

TAGS

A sample ruleset:

div{ text-align: right; }

  • A ruleset is made up of two parts:

    1. A Selector (div): which is the code that selects the HTML elements you want to style.
    2. Declaration block { #statements }: which is the code that describes the styles you want to apply to said HTML. Declarations are the lines of code written inside the curly braces.

  • The head contains the meta-data of the page. Style is one of the pieces of information that goes inside the head. Anything that goes inside the style tags is CSS.

  • Let us consider the set: p { colour: blue; }

    • p (tag selector) is telling the browser to select all the p tags, or paragraph tags of the page.
    • {colour: blue;} (declaration) colour is the property we wanna change and blue is the value we pass to it.
    • The value describes how the browser should render the property for the element. Properties are always written in plain English. And they identify a feature of an HTML element that we want to change.

    • NOTE THAT using p{} tag selector, we can only select ALL the paragraph tags. To specifically select a paragraph, we use attributes.
