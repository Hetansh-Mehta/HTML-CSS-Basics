CSS Stylesheets

Intorduction

A stylesheet is a file containing the code that describes how elements on your webpage should be displayed. It is similar to styling the HTML document, except the CSS lives in a different file. So we now don't have to use the style tags anymore.

To create a stylesheet, simply add a new file to your project, write some CSS, and save it as "name-of-stylesheet.css"

Link to a Stylesheet

Before the webpage can use the stylesheet, we need to link it to the HTML file. To create a link, we simply type the following inside the "head" of the HTML file:

link href="path-to-stylesheet/stylesheet.css" rel="Stylesheet"

The "href" attribute specifies the path to the linked resource and the "rel" attribute names the relationship between the resource and the document. In this case, the relationship is a stylesheet.
