CSS Units of Measurement

In CSS, we use units like pixels and percentages for measurement. Furthermore, Units can be categorised into two parts:

  • Absolute Units

    Units that refer to fixed units of measurement. Example: pixels, mm, in, cm

  • Relative Units

    Units that are a comparison to another linked property. Example: percentages, em, vw (viewport width), vh(viewport height)

Adding Color to Websites

While writing CSS, we express color by specifying the amount of red, green, and blue light that should be present. This can be done in two ways:

  1. RGB Each amount of red, green and blue light is written as a decimal number between 0 and 255. 0 being the lowest value or no presence of light and 255 being the highest value or 100% presence of the light.

    Example: body{ background-color: rgb(255,0,255); }

    This will change the background color to magenta.

  2. Hexadecimal or hex Each amount of red, green, and blue light is written as a hexadecimal number between the values 00 and FF. 00, which is just 0 in decimal, and FF, which is just 255 in decimal

    Example: body{ background-color: #ff00ff }

    The # sign tells the browser that hexadecimal is being used.
